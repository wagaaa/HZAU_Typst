# HZAU_Typst
Typst template for HZAU dissertation

华中农业大学学位论文Typst模版（开发中）

## 使用方法

1. 安装[Typst](https://typst.app/)
2. 安装字体依赖
3. 使用推荐的编辑器（VSCode）编辑main.typ
